# SCT_DS_Task04

## Task Description
This task focuses on analyzing traffic accident data to identify patterns based on
road conditions, weather conditions, and time of day.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Analysis Performed
- Data loading and preprocessing
- Time-based accident analysis
- Visualization of accidents by road condition and weather

## Conclusion
The analysis shows that road and weather conditions significantly impact accident
occurrence. Wet roads and evening hours show higher accident frequency, highlighting
the importance of safety measures during such conditions.
